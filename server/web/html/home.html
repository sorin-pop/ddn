Cloud DB 2.0

The successor to the previous cloud database, this one is much more versatile and extensible.

On the right, you can see the active connectors; These are the available databases in which you can create and import databases.

Currently, the following database vendors are tested and supported:

  1. MySQL 5.5, 5.6, 5.7
  2. PostgreSQL 9.4 (not backend not yet created)
  3. Oracle 11g

There are plans to add support for SQL Server as well

Use the top nav to create or import a database.
