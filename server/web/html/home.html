Cloud DB 2.0

The successor to the previous cloud database, this one is much more versatile and extensible.

On the right, you can see the active connectors; These are the available databases in which you can create, and later, import a database.

Currently, the following database vendors are tested and supported:

  1. MySQL 5.5, 5.6, 5.7
  2. PostgreSQL 9.4
  3. Oracle 11g

There are plans to add support for SQL Server as well

To create a database, click on the "Create Database" link in the navbar - that will take you to a form.

In the form you can optionally add your choice for database name, user and password, however if you don't, that's also fine, the page will generate them for you.
